Lightweb Media Basic Authentication for protected your development WordPress site like .htpasswd

== Description ==

Lightweb Media Basic Authentication is a plugin for protected your development WordPress site like .htpasswd.
The Plugin supported an localhost bypass for the wp-cli and an mu-plugin creation for Rollbacks.

Features:

* Easy for setting Basic Authentication
* Basic Authentication works like .htpasswd
* Protected development website or demo website without .htpasswd
* Support Docker and Kubernetes (K8s)
* The plugin is lightweight.
* Create mu-plugin for rollbacks
* Allow Bypassing for localhost (for WP-CLI)
